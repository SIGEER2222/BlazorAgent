M4 蓝图市场 UI 原型

目标与路由
- 路由：`/factory/market`
- 目标：蓝图上传/浏览/评分与参数化展示。

数据源与契约
- `POST /api/market/blueprints` → `{ id, version }`
- `GET /api/market/blueprints?id=...` → `BlueprintShare`

商品市场页签（新增）
- 数据契约：
  - `GET /api/market/orders` → `MarketOrder[]`
  - `POST /api/market/trade` → 下单成交，返回更新后的预算与库存摘要
- 布局与交互：
  - 左侧订单列表（买/卖分组），支持筛选与排序（价格/数量/到期）。
  - 右侧预算卡片（余额/资产），下单表单（数量/价格/类型）。
  - 成交后弹出反馈与链接到库存/经济页面。
- 验收：
  - 成交更新预算与库存；到期订单自动移除或灰显。

布局与交互
- 列表页：蓝图卡片（名称、标签、评分、下载量）。
- 详情页：参数化预览、版本历史、安全校验结果。
- 操作：上传、评分、下载与部署（跳转蓝图部署页）。

验收标准
- 上传与版本治理正确；参数预览与部署联动。
